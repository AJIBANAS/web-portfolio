# 🚀 Aplikasi Web Blog Portfolio Peribadi

## 📌 Deskripsi Projek
Projek ini merupakan sebuah aplikasi web **Blog Portfolio Peribadi** yang dibangunkan khusus bagi memenuhi kriteria penilaian Tugasan Individu untuk kursus **CSD34203 (Project Portfolio)** di UniSZA. Pembangunan sistem ini memfokuskan kepada penggunaan sistem kawalan versi kod (*Version Control System*) berasaskan **Git** dan platform **GitHub**.

---

## 🛠️ Ciri-Ciri Utama & Teknologi
* **Reka Bentuk Responsif:** Antara muka web yang mesra pengguna dan optimum bagi paparan skrin komputer mahupun telefon pintar.
* **Tema Gelap Moden (Teal Dark Theme):** Mengurangkan keletihan mata menggunakan palet warna kelabu gelap dan biru firus (*teal*).
* **Aliran Kerja Git Sistematik:** Setiap fasa perubahan kod didokumentasikan dengan mesej komit (*commit message*) yang jelas.

### Teknologi Yang Digunakan:
* **HTML5** - Membina struktur logikal kandungan halaman web.
* **CSS3** - Menghias susun atur elemen (*styling*) dan kesan transisi animasi.
* **GitHub Desktop** - Menguruskan repositori dan penyegerakan fail (*synchronization*).

## ⚙️ Panduan Menjalankan Projek Secara Lokal
Untuk melihat atau menjalankan projek ini pada komputer anda, ikuti langkah mudah ini:

1. Klik butang **Code** pada halaman repositori GitHub ini dan pilih **Download ZIP** (atau gunakan arahan `git clone`).
2. Ekstrak fail tersebut ke komputer anda.
3. Cari fail bernama `index.html`.
4. Klik dua kali (*double-click*) pada fail `index.html` tersebut untuk membukanya secara terus menggunakan sebarang pelayar web pilihan anda (Google Chrome, Microsoft Edge, Safari).
