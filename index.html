👤 Mengenai Diri Saya

Hai! Saya seorang pembangun web yang kini sedang mengikuti kursus pembangunan perisian di Fakulti Informatik dan Komputer (FIK), UniSZA. Projek ini dibangunkan khas bagi memenuhi keperluan kriteria tugasan individu untuk kursus CSD34203 (Project Portfolio).

Fokus utama saya adalah untuk memahami aliran kerja industri sebenar dengan mengaplikasikan kaedah version control menggunakan Git dalam menguruskan perubahan kod aplikasi.
